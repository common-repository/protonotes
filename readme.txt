Adds Protonotes to your WP site

== Descriptiion ==

Adds Protonotes to your WP site (for logged in user only) and provides links to your Protonotes as RSS/Excel from the backend.

What is Protonotes?
Protonotes are notes that you add to your prototype that allow project team members to discuss system functionality, design, and requirements directly on the prototype. You can think of it like a discussion board/wiki in direct context of your prototype. See [Protonotes.com](http://www.protonotes.com/)

Features:
* Shows Protonotes to user which are logged in - not regular site visitors

== Installation ==

1. Upload the folder `protonotes` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert your Protonotes Code at Settinges > Protonotes (you will find a link to get your code there)

You're done!
